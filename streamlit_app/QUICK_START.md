# 快速开始指南

## 🚀 启动应用

### 方法一：使用启动脚本（推荐）

```bash
cd streamlit_app
./run.sh
```

### 方法二：直接运行

```bash
cd streamlit_app
pip install -r requirements.txt
streamlit run app.py
```

应用会自动在浏览器中打开：**http://localhost:8501**

## ✨ 核心特性

### 1. **实时更新** ⚡
- 调整左侧任何参数
- 结果**立即自动更新**
- **无需点击按钮**

### 2. **参数配置** ⚙️
左侧面板包含所有配置：
- **基础设置**：模拟天数、开始日期、额外投放支出
- **预算策略**：基准预算比例、6个地区分配
- **全局默认参数**：初始 DAU、CPI、ARPU、运营成本、自然量增长
- **留存率配置**：7个关键节点（4位有效数字精度）

### 3. **结果展示** 📊
主区域显示：
- **关键指标卡片**：最终 DAU、累计收入、累计成本、净利润
- **趋势图**：DAU 和 DNU（自然/付费）趋势
- **P&L 曲线**：累计利润曲线，标注盈亏平衡点
- **地区对比**：各地区贡献度饼图

## 🎯 使用技巧

1. **实时调整**：拖动滑块或输入数值，结果立即更新
2. **地区分配**：确保总和为 100%（会有提示）
3. **留存率拟合**：点击"留存率拟合参数"查看拟合结果
4. **详细指标**：点击"详细指标"展开查看更多信息

## 📝 与 React 版本的区别

| 特性 | Streamlit 版本 | React 版本 |
|------|--------------|-----------|
| **实时更新** | ✅ 自动 | ❌ 需点击按钮 |
| **代码量** | ~500 行 | ~2000+ 行 |
| **启动** | 一个命令 | 两个服务 |
| **迭代** | 修改即生效 | 需要编译 |
| **部署** | 简单 | 复杂 |

## 🔧 故障排除

### 导入错误
如果遇到导入错误，确保在 `streamlit_app` 目录下运行：
```bash
cd streamlit_app
streamlit run app.py
```

### 端口被占用
如果 8501 端口被占用，使用其他端口：
```bash
streamlit run app.py --server.port=8502
```

## 📦 部署

### Streamlit Cloud（最简单）
1. 推送到 GitHub
2. 访问 https://streamlit.io/cloud
3. 连接仓库，自动部署

### 本地服务器
```bash
streamlit run app.py --server.port=8501 --server.address=0.0.0.0
```
